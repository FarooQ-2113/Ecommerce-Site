# :rocket: Features

- List Products
- Filter products by Category
- Add products to Wishlist
- Add products to Cart
  
# :construction_worker: How to run
## Install

```sh
npm install
```
## Usage

```sh
npm run start
```
## Dev Mode

```sh
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
